GOV.UK Design Guide example

To set up a template like this one

  1. Grab the gov.uk template
  2. Set up a base Sass stylesheet
  3. If not using Ruby or Node.js - include the govuk_frontend_toolkit repository as a git submodule with the /assets/ folder. Instructions here.
  4. At the top of the base Sass file, import the govuk_frontend_toolkit files. Instructions here.
  5. Create _vars.scss file for commonly used variables and import this next.
  6. Create styles from the gov.uk front-end mixins and variables
    1. Create a phase banner class
  7. Create additional styles for scoped typography from the front-end toolkit mixins
