Design guide example

This example provides a set of basic styles, to show how the mixins and variables within the front-end toolkit can be used to create content in the GOV.UK style.

Styles are applied using the .gov-uk class on the <main> content area.

Using Sass

This example assumes you are familiar with Sass and will use Sass to generate your stylesheets. Here, application.scss is the file that will be compiled with Sass.

Take a look at the application.scss to see how this example project is structured.

The basic.scss stylesheet is structured in a mobile-first fashion. Mobile styles are defined first, then the media mixin is used to override the default mobile styles with tablet or desktop styles.
