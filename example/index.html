GOV.UK design guide example

This page uses the gov.uk template, gov.uk front-end toolkit and additional template styles.

How to set up a template like this one:

  1. Grab the gov.uk template
  2. If not using Ruby or Node.js - include the govuk_frontend_toolkit repository as a git submodule with your /assets/ folder. Instructions here.
  3. Set up a base Sass stylesheet within /assets/sass/
  4. At the top of the base Sass file, import the govuk_frontend_toolkit files. Instructions here.
  5. Create a folder /yourprojectname/ for your project, here we are using /example/
  6. Create a _vars.scss file within /yourprojectname/ for commonly used variables and import this next.
  7. Import the gov.uk _template.scss file within /yourprojectname/ for basic typography and layout.
  8. Create a /design-patterns/ folder within your /sass/ folder and create modules for additional styles as required.

The base stylesheet /assets/base.scss should look like this:


// Design Guide example stylesheet

// Import the frontend toolkit
@import "../govuk_toolkit/stylesheets/colours";
@import "../govuk_toolkit/stylesheets/conditionals";
@import "../govuk_toolkit/stylesheets/css3";
@import "../govuk_toolkit/stylesheets/device-pixels";
@import "../govuk_toolkit/stylesheets/measurements";
@import "../govuk_toolkit/stylesheets/shims";
@import "../govuk_toolkit/stylesheets/typography";

@import "../govuk_toolkit/stylesheets/design-patterns/alpha-beta";
@import "../govuk_toolkit/stylesheets/design-patterns/buttons";

// Where 'example' is your project name

// Import variables
@import "example/vars";

// Import template (typography & layout) base styles
@import "example/template";

// Import design patterns (modules)
@import "design-patterns/alpha-beta";
@import "design-patterns/breadcrumb";
@import "design-patterns/buttons";
@import "design-patterns/footer";
@import "design-patterns/service-name";

// Import additional modules as required
