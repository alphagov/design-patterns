About this guide

This guide aims to help achieve a consistent look across government services.

Typography

Services should use clear, easy to read type, with consistent styles and a clear hierarchy of information.

Driving and transport
View HTML & SCSS for this example

Using the heading-48 mixin, with additional styles for spacing, using the $gutter-half variable = 15px.

Usage: 

            
              <div class="page-header">
                <h2 class="heading-48">Driving and transport</h2>
                <p class="heading-24">
                  Book your driving test or get a tax disc online, 
                  find out the legal requirements for buying, owning, 
                  importing or scrapping a car or motorcycle.
                </p>
              </div>
              
              .page-header {
                 padding: $gutter-half 0;
              }
              .page-header .heading-48 {
                 @include heading-48;
                 font-weight: 700;
                 padding-top: 26px;
                 padding-bottom: 14px;
              }
              .page-header .heading-24 {
                 padding-top: $gutter-half;
              }

Images

Avoid using images, GOV.UK is for services and information, not photographs. If images cannot be avoided they should be landscape, 3:2 aspect ratio.

Buttons

Primary action buttons are green, secondary action buttons are grey, irreversible action buttons are red.

Button Button Button
View HTML & SCSS for this example

The front-end toolkit provides button mixins in _buttons.scss.

Usage: 

            
              <button class="button-primary">button text</button>
              
              .button-primary {
                @include button($green);
                @include core-19;
              }

Here we also include the core-19 mixin to set the button font size to 19px.

Github: GOV.UK front-end toolkit - buttons
