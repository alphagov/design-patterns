Headings

Sizing your text appropriately for the column width should be your starting point. Use 48px as the recommended heading size and reduce where titles are too long.

The following heading sizes are commonly used.

48px heading (page titles)
36px heading (content page titles)
36px heading (page sections)
27px heading (intro or lead text)

Page titles

Where possible, page titles should use 48px bold text. Where heading text is too long for 48px bold text, reduce the size to 36px bold.

Page titles have plenty of white space surrounding them. 60px above and below for 48px text, 45px above and below for 36px text.

Body text

Type should be large enough to be easily read. This is generally larger than many current websites: 36px for headlines, 19px for body text. Leading body text - for example an intro paragraph, can be larger at 24px.

Colour

Generally we use type in #0B0C0C against a white or light grey background. Links should be blue and underlined.

Text and link colours

Text must have enough contrast against the background colour to be readable. This should be tested to conform with our Accessibility requirements.

Images

Avoid using images, GOV.UK is for services and information, not photographs. If images cannot be avoided they should be landscape, 3:2 aspect ratio

Buttons

Buttons should have a default colour (such as green). Alternative colours should be possible but only to indicate different actions. Button text size should be the same as body text, for desktop this is 19px.

Grid layout

Tablet or desktop layouts use either a three or four column fluid grid. Both grids can be present on a single page.

Narrower layouts intended for mobile devices or older web-browsers might find it simple to use a single column.

The gov.uk frontend toolkit provides width variables for common grid layout widths.
